# Agent Skills

Shared skills live here using the Agent Skills layout expected by `gh skill`.

`gh skill` is in preview, so prefer checking the current CLI help for validation, installation, and publishing commands:

```bash
gh skill --help
```

## Skills

- `html-briefing` - Render a plan, PR review, explanation, report, comparison, or lightweight local artifact as a self-contained HTML page and open it in the browser.
