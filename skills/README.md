# Agent Skills

Shared skills live here using the Agent Skills layout expected by `gh skill`.

`gh skill` is in preview, so prefer checking the current CLI help for validation, installation, and publishing commands:

```bash
gh skill --help
```

## Skills

- `diff-refine` - Iteratively refine an existing code or config diff with reviewer and implementer agents.
- `tmd-markdown` - Present Markdown files in the terminal with `tmd`.
